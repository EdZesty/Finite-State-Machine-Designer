Instrucciones de Uso

- Doble clic en el canvas para agregar un estado.
- Shift + arrastrar para crear una transición entre estados.
- Clic derecho en un estado para marcarlo o desmarcarlo como estado de aceptación (solo puede haber un estado de aceptación).
- Ingresa una cadena en el campo de texto y presiona "Validate" para verificar si la cadena es aceptada por el autómata.
- Arrastra un estado para moverlo.
- Mantén presionada la tecla "Suprimir" y haz clic en un estado o transición para eliminarlo.
- Presiona "Reset Diagram" para borrar todo el diagrama y empezar de nuevo.
